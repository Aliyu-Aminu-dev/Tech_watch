📖 Vue Blog Platform
A simple, responsive, and performant blog application built with Vue.js.
This project demonstrates routing, state management, API integration, testing, accessibility, and CI/CD deployment.

🚀 Features
Multi‑page routing with Vue Router (Home, Blog, Post, About).

Global state management using Pinia.

API integration with JSONPlaceholder.

Responsive design using Tailwind CSS.

Performance optimisations: lazy loading, code splitting, image optimisation.

Testing with Vitest and Cypress.

CI/CD deployment via Vercel or Netlify.

Accessibility audit with Lighthouse.

📂 Project Structure
Code
src/
├── assets/        # images, icons
├── components/    # reusable UI (Navbar, Footer, PostCard)
├── pages/         # Home.vue, Blog.vue, Post.vue, About.vue
├── store/         # Pinia stores
├── router/        # Vue Router setup
├── App.vue
└── main.js
🛠️ Tech Stack
Framework: Vue 3 + Vite

Routing: Vue Router

State Management: Pinia

Styling: Tailwind CSS

API: JSONPlaceholder

Testing: Vitest + Cypress

Deployment: Vercel / Netlify

Accessibility: Lighthouse, ARIA labels

⚡ Getting Started
1. Clone the repo
bash
git clone https://github.com/your-username/vue-blog-platform.git
cd vue-blog-platform
2. Install dependencies
bash
npm install
3. Run locally
bash
npm run dev
4. Build for production
bash
npm run build
🧪 Testing
Run unit tests:

bash
npm run test
Run end‑to‑end tests:

bash
npm run cypress:open
🌍 Deployment
Push to GitHub.

Connect repo to Vercel or Netlify.

Automatic deployment on every commit.
